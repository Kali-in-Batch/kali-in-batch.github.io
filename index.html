The Kali in Batch project

Welcome to the official website for the Kali in Batch project. Here you can find the software made by the project, and useful information about the project.

Software

The Kali in Batch project has made the following software:

Kali in Batch, a Kali Linux-like environment written in Batch and PowerShell, extensible by packages.

Package repository for the Kali in Batch software.

Linux on Batch on Windows, an experimental software to run Linux x86_64 NASM on Windows natively.

Mission of the project

The mission of the project is to create a Linux-like environment for Windows, using Batch and PowerShell.
